# ScreenForge Landing

This folder contains the marketing/landing site for ScreenForge, built with Next.js (App Router) and Tailwind CSS.

## Requirements

- Node.js (LTS recommended)

## Getting started

From this folder:

```bash
cd screenforge-landing
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 to view the site.

## Build

```bash
npm run build
npm run start
```
